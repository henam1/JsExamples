What is JavaScript and what can it be used for?

JavaScript is a cross-platform programming language that runs in a host environment like a Web browser. The majority of websites use JavaScript and all Web browsers support JavaScript without any need of plug-ins. JavaScript is used to make a website interactive, one example are polls that can be found on many websites

Is JavaScript and Java the same thing?

Even though there are some similarities in syntax between JavaScript and Java they are fundamentally two different programming languages. JavaScript is in some ways a bit less structured programming because it is not necessary to declare all variables, classes and methods. Also return types are not explicitly typed.

Do I need to know how to program JavaScript to be able to use it on my website?

No, you do not have to know how to program JavaScript to be able to use some cool scripts to use for your website. On this page you will find some examples that you can use straight away and implement straight into your website.
